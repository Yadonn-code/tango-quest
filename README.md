# Jと戸塚からの体罰を避けろ ～理性は悪なんよ～

英単語の4択クイズで5つのステージを冒険するRPG風の単語テストアプリ。
まちがえると **J** と **戸塚** の体罰が飛んできてHPが減り、HPが尽きるとセーブポイントまで戻されます。

ブラウザだけで動く純粋な HTML / CSS / JavaScript 製。インストール不要です。

## 遊び方

1. 上のURLを開く（またはダウンロードして `index.html` をブラウザで開く）
2. 難易度を選ぶ
3. 各ステージ10問の4択クイズに答えて、全5ステージの制覇を目指す

## ルール

- 1ステージ = 10問。全5ステージ（草原 → 森 → 洞窟 → 雪山 → 魔王城）
- まちがえるとランダムで J か戸塚に殴られる（戸塚のほうがダメージ大）
- HP は 100。0 になるとセーブポイント🚩まで戻される
- セーブポイント到達で HP 全回復＆進行状況をブラウザに保存（「つづきから」で再開可能）
- セーブポイント以外のステージクリアでは HP が 20 だけ回復

### 🔥 コンボ

- 連続正解でコンボが伸びる（HUDに表示）。まちがえるとリセット
- **5コンボごとに気合いでHPが10回復**
- 最大コンボはエンディングの成績と称号に反映される

### 🍙 おにぎり（回復アイテム）

- ステージクリアで1個入手（最大3個持てる）
- クイズ中にHUDの🍙ボタンで使用 → HPが30回復

### ⚔️ ボス戦（ステージ5・魔王城）

- 最終ステージはクイズバトル。**正解すると敵にダメージ**を与えられる
- 中ボス **J**（HP50）→ 魔王 **戸塚**（HP100）の連戦
- コンボが伸びるほど攻撃力アップ。5コンボ以上で **💥会心の一撃**
- まちがえるとそのボスから体罰が飛んでくる。倒されるまで出題は続く

### 🎵 BGM

- 冒険中はフィールド曲、魔王城はボス戦曲、クリア後はエンディング曲が流れる
- 音声ファイル不要（ブラウザのWebAudioでチップチューンをリアルタイム生成）
- タイトル画面の「🔊 BGM」ボタン、またはクイズ中のHUD右上の🔊でON/OFF（設定は保存される）
- スマホ対応: ブラウザの仕様上、**最初に画面をタップしてから**音が鳴り始める。iPhoneのマナーモード中でも鳴るように対策済み（音を消したいときは🔊ボタンでOFFに）

### 👊 だれの攻撃かわかる表示

- 体罰が飛んでくるとき、画面上部に「😎 Jのこうげき！」（青）／「👹 戸塚のこうげき！」（赤）のバナーが出る
- ボス戦で自分が攻撃するターンは「🗡️ あなたのこうげき！」（黄）

## 難易度

| 難易度 | 😎 J のパンチ | 👹 戸塚のパンチ | 🚩 セーブポイント | ⏰ 時間制限 | 激ムズ単語 | 主人公の攻撃力 |
| --- | --- | --- | --- | --- | --- | --- |
| 🌱 イージー | -10 | -20 | 全ステージ | なし | なし | 100% |
| ⚔️ ノーマル | -15 | -35 | ステージ 1・3・5 | なし | なし | 100% |
| 🔥 ハード | -30 | -60 | ステージ 1 のみ | **1問10秒** | 出題の約6割 | 65% |
| 💀 エキスパート | -40 | -80 | なし（死んだら最初から） | **1問6秒** | 出題の約9割 | 50% |

ハード・エキスパートでは1問ごとにカウントダウンがあり、時間切れは不正解あつかいで体罰が飛んできます。さらに敵の攻撃力が高いだけでなく主人公自身の攻撃力も抑えめになっており、「打たれ弱いのに火力も出ない」高難易度に調整されています。

### 💀 エキスパート（隠し難易度）

- **ハードをクリアすると解放**されます（初回クリア以降、難易度選択画面で選べるようになります）
- 出題される単語のほとんどが [words.js](words.js) の `HARD_WORDS`（ネイティブでも辞書必須クラスの語彙）から出題されます
- ステージ5の最終決戦で中ボスJ・魔王戸塚を倒したあと、**さらに裏ボス2連戦**が待っています。倒すたびに正体が明らかになるのでお楽しみに

## 📖 一問一答モード

タイトル画面の「一問一答モード」から入れる、体罰なしのシンプル暗記モードです（[drill.html](drill.html)）。

- 英単語を見て意味を思い浮かべたら「こたえを見る」→ 自分で ⭕／❌ をつける
- 出題数は 10問／25問／50問／エンドレスから選択（エンドレスはいつでも終了OK）
- 結果画面に ❌ だった単語の一覧が出て、その単語だけ復習できる
- キーボード対応：スペース＝こたえを見る、1＝⭕、2＝❌

## 単語リストのカスタマイズ

出題される単語（大学受験レベル 約1340語）は [words.js](words.js) の `WORDS` にまとまっています。

```js
const WORDS = [
  { en: "consider", ja: "〜とよく考える" },
  // ← この形式で自由に追加・編集できます
];
```

ハード・エキスパート専用の激ムズ単語（約120語）は同じファイルの `HARD_WORDS` にまとまっており、同じ形式で追加・編集できます。

- 英単語以外（古文単語・歴史用語など）に差し替えても動きます

## キャラ画像のカスタマイズ

J と戸塚の顔はオリジナルのSVGイラストです。自作イラストなど権利上問題のない画像に差し替えたい場合は、画像ファイルをこのフォルダに置いて `game.js` の `CUSTOM_FACES` にファイル名を指定してください。

```js
const CUSTOM_FACES = { j: null, totsuka: "totsuka.png" };
```

## ローカルで動かす

ダウンロードして `index.html` をダブルクリックするだけで動きます。

## ファイル構成

- `index.html` — 画面構成（クエストモード）
- `drill.html` / `drill.js` — 一問一答モード
- `style.css` — 見た目・アニメーション
- `game.js` — ゲームロジック（ステージ・ダメージ・セーブ処理）
- `words.js` — 単語データ
